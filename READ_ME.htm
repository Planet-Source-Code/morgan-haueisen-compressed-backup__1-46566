Compressed Backup

 

This is not meant to be a replacement for WinZip nor is it WinZip compatible (does not have all the header information).  Its sole purpose is to provide your application a way to backup its data files in a compressed format.  It is one class; you supply the source path and file name, destination path and file name, and the action (compress or decompress).

It only compresses one file, so if you have multiple file to back-up, each will be compressed individually.

 

When decompressing a file, if you do not supply a file extension, the original file extension will be used.  It has some checks for free disk space and for a valid compressed file.

 

It does require zlib.dll, which needs to be copied to your windows/system directory.  It can be downloaded FREE from http://www.winimage.com/zLibDll/  by clicking on the zlib114dll.zip link.  It does not need to registered or installed.

 

The 32bit zlib.dll (V1.1.4) compression library was originally written by Jean-loup Gailly and Mark Adler (variation of LZ77 Lempel-Ziv 1977 algoritm).
